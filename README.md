# VirusCity

Telegram Mini App — браузерная PvP-игра, где ты мэр города во время вирусной вспышки. Строишь город, сдерживаешь эпидемию, атакуешь соседей своим штаммом.

## Возможности

- Интерактивная сетка города 8×8 — строй и улучшай здания по кварталам
- 3 штамма вируса: Альфа (медленный, заразный), Дельта (быстрый, дорогой), Омега (мутирует непредсказуемо)
- Три решения мэра: карантин, вакцина или игнорировать — каждое с экономическими последствиями
- Уровни заражения жителей: здоровый → полузомби → зомби
- Офлайн-прогресс — вирус распространяется пока ты не в игре, при входе видишь последствия
- PvP-атаки: запусти свой штамм в город соседа, тот получит уведомление в Telegram
- Альянсы с иммунным буффом — чем больше участников, тем сильнее защита
- Таблица лидеров топ-50

## Стек

- **Frontend**: React + TypeScript + Vite + Pixi.js + Zustand
- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL
- **Cache**: Redis
- **Bot**: node-telegram-bot-api
- **Deploy**: Vercel (frontend) + Render (backend)
